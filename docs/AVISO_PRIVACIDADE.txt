AVISO DE PRIVACIDADE — Sistema de Gestão de Ativos de TI
═══════════════════════════════════════════════════════════════════════════════

Este aviso está disponível sob Lei 13.709/2018 (Lei Geral de Proteção de Dados).

───────────────────────────────────────────────────────────────────────────────
1. DADOS PESSOAIS TRATADOS
───────────────────────────────────────────────────────────────────────────────

O sistema trata os seguintes dados pessoais:

• Nome do usuário
• Endereço de email corporativo
• Endereço IP (apenas para segurança e auditoria)
• Data e hora de acesso
• Ações realizadas no sistema (auditoria)

───────────────────────────────────────────────────────────────────────────────
2. FINALIDADE DO TRATAMENTO
───────────────────────────────────────────────────────────────────────────────

Os dados são tratados exclusivamente para:

✓ Autenticação e controle de acesso
✓ Gestão e rastreamento de ativos corporativos
✓ Segurança do sistema e detecção de uso indevido
✓ Cumprimento de obrigações legais
✓ Investigação de incidentes de segurança

───────────────────────────────────────────────────────────────────────────────
3. BASE LEGAL
───────────────────────────────────────────────────────────────────────────────

Tratamento autorizado sob:

• Artigo 7°, II da LGPD: Cumprimento de obrigação legal
• Artigo 7°, VI da LGPD: Legítimo interesse da empresa
• Artigo 17 da LGPD: Contexto de relação contratual/laboral

───────────────────────────────────────────────────────────────────────────────
4. RETENÇÃO DE DADOS
───────────────────────────────────────────────────────────────────────────────

Período de retenção por tipo:

• Dados de usuário ativo: enquanto ativo + 30 dias após desativação
• Logs de auditoria: 90 dias (exceção: logs de segurança, 180 dias)
• Hashes de senha: armazenado permanentemente (dados técnicos)
• Dados de recuperação: 24 horas (tokens expiram automaticamente)

───────────────────────────────────────────────────────────────────────────────
5. SEGURANÇA
───────────────────────────────────────────────────────────────────────────────

Medidas técnicas de proteção:

✓ Senhas armazenadas em hash irreversível (PBKDF2 + pepper)
✓ Transmissão segura via HTTPS
✓ Cookies com proteção HTTPONLY e SECURE
✓ Isolamento de dados por empresa
✓ Controle de acesso por perfil
✓ Rastreabilidade completa de ações

───────────────────────────────────────────────────────────────────────────────
